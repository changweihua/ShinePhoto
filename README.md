ShinePhoto
==========

基于 Caliburn.Micro 的 WPF 应用程序开发实践

## ShinePhoto ##

> 基于 Caliburn.Micro 的 MVVM 设计模式 <br />
> 使用 Sqlite 数据库存储配置信息 <br />
> 采用 RSA 和 椭圆签名算法实现软件的注册 <br />
> 引用 Extended WPF Toolkit 社区版，实现输入的定制化 <br />
> <br />
> <br />
> <br />
> <br />
